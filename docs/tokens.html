Design tokens

Design tokens are named entities that store visual design information. These are used in place of hard-coded values (such as hex values for color or pixels for spacing) in order to maintain a scalable, consistent system for UI development.

Borders

Width and radius tokens are used to style element borders. Accessed via map(borders, <key>).

Width

Accessed via map(borders, width, <key>).

Key Value
thin 1px
thick 2px
thicker 4px
thickest 8px

Radius

Accessed via map(borders, radius, <key>).

Key Value
small 0.125rem
default 0.25rem
circle 50%
